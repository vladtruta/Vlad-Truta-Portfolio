Student-in-Service Calendar Generatormore_vert

Don't worry about dirty blackboards anymore!


Final Project for my High School CS Class

Student-in-Service Calendar Generatorclose

The Student-in-Service is a very common concept in Romanian schools. Basically, the students who are in service that day, are the ones responsible with cleaning the blackboards and keeping the classroom clean and tidy.

This app does just that! You insert the students in any order you wish, either from the software built-in menus or from an external text file, and it automatically generates an Excel Spreadsheet on a set period of time, in a real calendar (usually it’s best to set full months).

I developed it in order to fix a common problem we had in our classroom. Every month, someone had to create a spreadsheet with all the students, so we would know who is going to be the one in service that day. And they had to re-write this spreadsheet over and over, every month, in order to keep the correct rotation of the students.

Largonjimore_vert

Decode words and learn the long-lost French jargon!


Winner of the 2nd Place at the Third Edition of Prix CFA

Largonjiclose

This game was made for the Prix CFA Contest, Third Edition.

Largonji is a derivative of the French jargon louchébem. The louchébem word-creation process resembles that of verlan, and javanais, in that existing words are camouflaged according to a set of rules.

Largonji substitutes ⟨l⟩ for the consonant or consonant cluster at the beginning of the word, or, if the word begins with an ⟨l⟩ or a vowel, the second syllable; the initial consonant is then reattached to the end of the word along with a suffix particular to the argot: -ji [ʒi], -oc [ɔk], -ic [ik], -uche [yʃ], -ès [ɛs], or in the case of louchébem, -em/ème

The word database is very big, and the cool thing about it is that it's hosted on my server. Everytime the game is launched, an API will keep it up to date.

The game also has an option in which you can suggest new words to be added to the list. Facebook Integration is there, but still has some issues.

Goal-Keeper is a Prototype for a Desktop Application, which helps you organize your schedule and achieve a life goal.

Mobile VoiceInvoicemore_vert

Use your voice to fill invoices!


Android App Prototype from the MegaHack 2016 Hackathon

Mobile VoiceInvoiceclose

This is the VoiceInvoice app prototype that we built for Life is Hard at the TechFest MegaHack hackathon. It generates an invoice using voice command.

The application is easy to understand. It starts with an interface made up of input fields. Periodically, you will hear a beep. It usually happens every few seconds, though it does not have a set interval.

Right after the beep, you say a voice command. All voice commands start with “okay”, in order to distinguish a voice command from regular text.

We implemented a few voice commands for navigation, such as “okay next”, or “okay back”, and others for text processing.

MyTradeMarketmore_vert

Your own private trade market!


Desktop Application Prototype from the PoliHack v3 Hackathon

MyTradeMarketclose

MyTradeMarket is the app we created for the PoliHack v3 hackathon.

MyTradeMarket is a private trade room prototype app. Just like private chat rooms, where you connect via IP address, our idea was to create the same thing, but oriented towards the selling/buying of products, and done privately.

The advantage of this is that you don’t have to enter your credentials on any website in order to sell your products. Just add them into your private store, and give your customers your IP address, and they will connect and be able to see your products. We thought that this idea would help those who want to sell products in a hassle-free environment that doesn’t spy nor track buyers and sellers.

The application was developed to work with Bitcoin, however it can be extended to work with about any type of currency.
